O que é o HTML:

O HTML é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website. Portanto, ele serve para dar significado e organizar as informações de uma página na web. Sem isso, o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos.

Estrutura básica do HTML:

(voltar)

O HTML é constituído por uma estrutura de três pilares, HTML, head, body.

  • HTML

    • A abertura e o fechamento da sigla Html significa a abertura e fechamento da página, que delimita o documento. Sendo assim, todas as demais tags da página devem estar nesse espaço.

  • Head

    • A sigla Head é o cabeçalho da página, contendo a tag Title que é onde damos o nome da nossa página.

  • Body

    • Entre a abertura e o fechamento da sigla Body é o espaço no qual deve estar contido o conteúdo visual da página (corpo do site). As demais tags que representam texto, botões etc. devem ser adicionadas nesse intervalo.

Tags, o que são?

(voltar)

As tags HTML são elementos usados ​para marcar e formatar os conteúdos: textual ou visual em documentos HTML. Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta e fechada.

Por exemplo: a tag de parágrafo é escrita como < p > e < /p >. Já para negritar pode ser usada a tag < Strong > ou < b > no início e < /Strong> ou < /b > para finalizar.

Para finalizar a utilização de uma tag pode ser usada uma barra (/), sendo que no caso das tags de autofechamento, não há necessidade da presença desse caractere.

Principais tags de texto:

(voltar)

  • Título

    • A tag < h1 >< /h1 > indica o título maior do seu texto

  • Sub título

    • A tag < h2 >< /h2 > indica um sub-título do seu texto, tendo outros sub-títulos titulos também, sendo eles do < h2 >< /h2 > até o < h6 >< /h6 >, quanto maior o número, menor é o tamanho do sub-título.

  • Parágrafo

    • Para escrever textos em parágrafos, utilizamos a tag < p >< /p >

  • Citação

    • É utilizado a tag < blockquote >< /blockquote >para adicionar uma frase de efeito, uma frase menor para se destacar abaixo de algum texto ou imagem.

  • Negrito

    • é utilizado a tag < strong >< /strong >para destacar uma palavra ou frase em negrito.

  • Itálico

    • É utilizado a tag < i >< /i >para destacar uma palavra ou frase em itálico.

  • Marcar

    • É utilizado a tag < mark >< /mark >para marcar alguma palavra ou frase importante do texto.

  • Sublinhar

    • É utilizado a tag < u >< / u > para sublinhar uma palavra ou frase.

Tags de Lista:

(voltar)

  • Ordenada

    • É utilizada a tag < ol >< /ol > para indicar uma lista ordenada também conhecida com lista pai.

  • atribuir itens

    • É utilizado a tag < li >< /li > também conhecida com lista filha para adionar itens na lista.

  • Não ordenada

    • É utilizada a tag < ul >< /ul > para uma lista não ordenada, uma lista ordenada pode ser composta por diversos marcadores, tanto bolinha, traços, estrela e etc.

Tags link:

(voltar)

No processo de criação de um site, quase sempre é utilizado links, seja direcionando à outras abas, páginas e sites ou ancorando na própria página.

Algumas tags se destacam no processo de criação de um link, sendo elas:

  • Criar link

    • Para a criação de um link é utilizado a tag < a href= "https://..." < /a >

  • Abrir link em outra pág

    • É utilizado a tag Target="self" para abrir na mesma aba ou a tag Target="blank"para abrir em outra aba

  • Ancorar link para onde cliquei

    • É utilizado a tag < href="#" > titulo do texto < /a > para direcionar automaticamente na matéria da página.

  • Criar balão na maozinha do link

    • É utilozada a tag Title="..."para criar um balãozinho ao encostar no link com uma mensagem indicando do que se trata.

  • Voltar para a pág anterior

    • É utilizada a tag < a href="..." >< voltar < /a > para voltar para uma página anterior
